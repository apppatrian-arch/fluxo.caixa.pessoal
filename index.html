Configurar GitHub

Os dados do app serão salvos no seu repositório GitHub. Configure uma vez e use em qualquer dispositivo.

  1. Acesse github.com → Settings → Developer Settings → Personal access tokens
  2. Clique em Generate new token (classic)
  3. Marque a permissão repo (acesso completo)
  4. Copie o token gerado e cole abaixo
